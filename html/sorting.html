    Venn Diagrams were first developed by John Venn in the 1880s. They are useful for illustrating the relationships between sets.
    Each set is illustrated by a circle. When two circles overlap, this shows an intersection of the two sets.
    In this activity, each of the properties of given objects are named like circles, red, etc. If we consider the Venn diagram with two sets and an intersection between them, then let each property be a variable, for example
    A = Red
    B = Circle
    There are four areas in the Venn Diagram, one which will contain only the objects which are red, one with only circles, the central area that has objects which are both red and circles, and everything which is not red, circle or both belongs to the universal set.
    Each of these four possibilities are denoted by an individual row in the truth table of 2 variables. So, each area of the Venn Diagram and each row of the truth table is called a "Valuation"/"Interpretation". The valuation holds true only in its corresponding area of the Venn Diagram.
